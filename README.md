# motec-i2

A MoTeC i2 file parser

Try the included examples with:
```
cargo run --example read
cargo run --example write
```

## Features

- [x] Parsing ld files
- [x] Writing ld files
- [ ] Parsing ldx files
- [ ] Writing ldx files
